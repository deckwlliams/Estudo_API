# Estudo_API

API REST feita com Java + Spring Boot para estudo de endpoints CRUD.

## Tecnologias
- Java
- Spring Boot
- Maven
- SQL

## Como executar
1. `git clone https://github.com/deckwlliams/Estudo_API`
2. `cd Estudo_API`
3. `mvn spring-boot:run`

## Funcionalidades
✔ Listar itens  
✔ Criar itens  
✔ Editar itens  
✔ Deletar itens
